# Mercaflash — Instrucciones para Claude Code

## Contexto del proyecto
Sitio web estático de Mercaflash (mercaflash.com.co), empresa fabricante de líquidos atmosféricos y comercializadora de productos de audio, iluminación y electrónicos. HTML/CSS/JS puro, sin frameworks ni dependencias externas de build.

## Stack
- HTML/CSS/JS vanilla (igual que SkyPets Colombia)
- Sin frameworks, sin build tools
- Deploy pendiente de definir (dominio: mercaflash.com.co)

## Estructura de archivos

## Tareas frecuentes
- **Actualizar landing:** editar `electro-landing.html`
- **Agregar producto:** buscar la sección de productos en el HTML y duplicar la card existente
- **Cambiar logo:** reemplazar `logo-mercaflash.webp` o `logo-mercaflash-dark.webp`
